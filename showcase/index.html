Flyout

Custom element "Flyout" with text that shows on the top of the parent element.

Call function ShowFlyout(element, string, delay).

  • element = Show flyout above this element.
  • string = Text to show in flyout.
  • delay = How long will flyout be visible (in milliseconds).

<button onclick="ShowFlyout(this, 'This is flyout', 2000);">Show flyout!</button>
