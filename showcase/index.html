Button

Customized button.

Normal button.

<button>Click me!</button> <!-- Can be used as <input type="submit"> --> <input type="submit" value="Click me!">

Accent button.

<button accent>I am an accent button</button>

Hyperlink button.

<button hyperlink>Hyperlink</button>

Textbox

Customized inputs with text.

Redesigned input type="text" with more attributes.

<!-- Attribute "clear_btn" shows clear text button. --> <input type="text" value="BeXCool" clear_btn> <input type="text" placeholder="Type here">

Redesigned input type="password" with more attributes.

<!-- Attribute "showpass_btn" shows button for showing password. --> <input type="password" value="1234" showpass_btn> <!-- Attribute "showpass" shows password. --> <input type="password" value="1234" showpass_btn showpass> <input type="password" placeholder="Type password">
